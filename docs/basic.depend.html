依赖加载

ndoojs可以为action定义依赖，当执行到action时会先加载需要的依赖(使用do.js加载)。do.js 参考do.js

Action依赖

ndoojs支持为单个Action定义依赖，格式为ActionName+Depend。示例：

// jquery模块定义
Do.define('jquery', {
  path: jsPathbase + '/jquery-2.1.1.min.js',
  type: 'js'
});
// 设置jquery为默认已加载
Do.setLoaded(['jquery']);
// 声明依赖
_n.app('home', {
  indexDepend: ['jquery'], 
  indexAction: function() {
    console.log(typeof jQuery);
  }
});

App依赖

可以直接为一个App定义依赖，在App中定义一个名为depend的属性即可。示例：

_n.app('home', {
  depend: ['jquery'], 
  indexAction: function() {
    console.log(typeof jQuery);
  }
});
