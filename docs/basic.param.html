参数的使用

1. page id中设置参数

<div id="scriptArea" data-page-id="home/index?abc">

2. 获取参数

indexAction: function(params){
  var $content;
  $content = $('#content');
  $content.html('module: home, action: indexAction param: ' + params);
}

此处params为字符串abc，ndoojs自运过滤掉了?（问号）
